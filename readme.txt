Atalki, a No Code web platform , is the fastest way to create a FREE personalised, shareable, smart FAQ webpage from text document within seconds.

== Description ==

This plugin helps you to generate any website page faq.

You can place the shortcode in any page and show it generated faq content in any page. The shortcode is: **[BZ-FAQ]**

You need to sign up first to generate faq content.

Then after you need to login with same credentials.



== Installation ==

e.g.

1. Upload `atalki-faq-generator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
